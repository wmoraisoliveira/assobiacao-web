# assobiacao-web

Landing page do Assobiacão — o primeiro app de torneios de assobio do Brasil.

## Estrutura

```
index.html        — landing page (HTML + CSS + JS, single file)
assets/
  mascot.png      — mascote (Lulu da Pomerânia)
  icon.png        — ícone do app
  favicon.png     — favicon
```

## Deploy

O projeto é um HTML estático — pode ser hospedado no GitHub Pages, Vercel, Netlify ou qualquer CDN.

### GitHub Pages
Vá em **Settings → Pages → Branch: main / root** e salve.

### Vercel / Netlify
Aponte para o repositório e deixe o build command vazio (static site).
